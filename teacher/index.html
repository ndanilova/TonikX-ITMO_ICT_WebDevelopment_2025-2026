    Teacher Dependencies

    Модель Teacher¶

    Основные характеристики¶

    Поля модели: - user - OneToOne связь с User моделью (является primary key) - teacher_number - уникальный идентификатор учителя (максимум 20 символов) - subjects - ManyToMany связь с Subject моделью (предметы преподавания)

    Ключевые методы¶

    Управление предметами и классами¶

    • get_taught_subjects() - возвращает все преподаваемые предметы
    • get_teaching_classes() - возвращает все классы, где учитель преподает
    • get_assigned_homeworks() - возвращает все домашние задания, назначенные учителем
    • get_subject_homeworks(subject) - возвращает задания по конкретному предмету

    Управление проверкой работ¶

    • get_ungraded_submissions() - возвращает все непроверенные работы
    • get_homework_submissions(homework) - возвращает все сабмишны по конкретному заданию
    • get_today_submissions() - возвращает работы, сданные сегодня
    • can_grade_submission(submission) - проверяет возможность оценки сабмишна
    • can_grade_homework(homework) - проверяет доступ к домашнему заданию

    Статистика и аналитика¶

    • get_homework_statistics(homework) - возвращает статистику по заданию
    • get_students_count() - возвращает общее количество студентов в классах учителя

    Teacher Views функциональность¶

    Вспомогательные функции¶

    • get_group_metrics(teacher, group) - вычисляет метрики производительности класса
    • get_subject_metrics(teacher, subject) - собирает данные по предмету
    • get_all_submissions(teacher) - получает все сабмишны учителя

    Профиль учителя (teacher_profile)¶

    • Отображает статистику по предметам и классам
    • Показывает сегодняшние сабмишны
    • Выводит метрики производительности классов
    • Отображает данные по каждому предмету

    Записи учителя (teacher_records)¶

    • Показывает все преподаваемые предметы
    • Отображает фильтры по предметам
    • Предоставляет доступ к детальной статистике

    Страница предмета (teacher_subject)¶

    • Детальная информация по конкретному предмету
    • Список классов, где преподается предмет
    • Навигация к классам предмета

    Страница заданий (teacher_assignments)¶

    • Полный список всех сабмишнов
    • Расширенная система фильтрации (по предмету, статусу, дате)
    • Сортировка по различным параметрам
    • Пагинация результатов

    Детали задания (teacher_assignment_detail)¶

    • Детальный просмотр сабмишна студента
    • Функциональность оценки работы
    • Система обратной связи
    • Валидация оценок

    Страница класса (teacher_school_class)¶

    • Детальная информация по классу и предмету
    • Статистика по каждому студенту
    • GPA студентов по предмету
    • Статусы выполнения заданий
    • Прямой доступ к сабмишнам для оценки

    Особенности реализации¶

    • Проверка прав доступа - учитель может оценивать только свои предметы
    • Комплексная статистика - расчет метрик производительности классов
    • Гибкая фильтрация - многоуровневая система фильтров и сортировки
    • Оптимизация запросов - предварительная загрузка связанных данных
    • Валидация данных - проверка корректности оценок и обратной связи
