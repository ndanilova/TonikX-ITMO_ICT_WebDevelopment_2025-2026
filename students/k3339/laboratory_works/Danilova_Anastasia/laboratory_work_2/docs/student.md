## Модель Student
### Основные характеристики

**Поля модели:**
- `user` - OneToOne связь с User моделью (является primary key)
- `student_number` - уникальный идентификатор студента (максимум 20 символов)
- `birth_date` - дата рождения студента
- `school_class` - ForeignKey к классу обучения с related_name='students'

### Ключевые методы

#### Академическая информация
- `get_all_subjects()` - возвращает все предметы класса студента
- `get_subject_homeworks(subject)` - возвращает домашние задания по конкретному предмету
- `get_subject_grades(subject)` - возвращает все оценки по предмету

#### Система оценок
- `get_average_grade(subject)` - вычисляет средний балл (по всем предметам или конкретному)
- `get_subject_gpa_with_zeros(subject)` - вычисляет GPA с учетом нулей за несданные работы
- `get_subject_records(subject)` - возвращает словарь {домашняя работа: оценка}

#### Управление заданиями
- `get_submitted_homeworks()` - возвращает все сданные работы
- `get_unsubmitted_homeworks()` - возвращает все несданные работы
- `get_pending_homeworks()` - возвращает работы с неистекшим сроком
- `get_overdue_homeworks()` - возвращает просроченные работы
- `get_burning_homeworks()` - возвращает "горящие" задания (срок до 7 дней)

#### Статусы и проверки
- `get_homework_status(homework)` - определяет статус задания ('graded', 'submitted', 'pending', 'overdue')
- `get_homework_grade(homework)` - возвращает оценку за задание (0 если не сдано)
- `get_homework_submission(homework)` - возвращает объект сабмишна если существует

### Student Views функциональность

#### Профиль студента (student_profile)
- Отображает статистику по предметам
- Показывает количество активных, "горящих" и просроченных заданий
- Проверяет наличие профиля студента

#### Академические записи (student_records)
- Отображает оценки по всем предметам
- Показывает GPA по каждому предмету
- Считает количество сданных работ

#### Детали предмета (student_subject)
- Детальная информация по конкретному предмету
- Все оценки и записи по предмету
- Статистика сдачи работ

#### Управление домашками (student_homeworks)
- Просмотр деталей задания
- Редактирование/сдача работы через модальное окно
- Валидация текста при отправке
- Сброс оценки при редактировании сданной работы

### Особенности реализации
- **Ролевой доступ** - проверка user.role == "student"
- **Обработка исключений** - корректная работа при отсутствии профиля
- **Динамическая статистика** - реальное время подсчета заданий
- **Гибкая система оценок** - разные методы расчета GPA
